

Projet : déploiement d’une instance EC2 via OpenTofu, intégrée dans un pipeline GitLab CI.

Fonctionnalités principales :

Provisioning automatisé d’une VM EC2 avec OpenTofu

Installation de Nginx sur la VM via un script shell

Déploiement d’une image Nginx personnalisée pour la VM

Sécurité et révision automatique de l’image GitLab grâce à Plumber > https://github.com/opengovsg/plumber.git une solution OpenSource decouvert au cours des Native days.

Objectifs :

Simplifier le processus de création et de configuration d’une VM

Assurer que l’image Nginx déployée est sécurisée et maintenue à jour

Intégration fluide avec GitLab CI/CD pour automatisation complète
